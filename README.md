# 🎨 AI Image Studio

An AI-powered image generation web application built using **Python** and **Streamlit** during the **MirAI School of Technology – Virtual Summer Internship 2026 (AI Builder Track)**.

Generate stunning AI images from text prompts with customizable image dimensions, multiple art styles, AI prompt enhancement, and a surprise prompt generator.

---

## 🚀 Features

- 🎨 Multiple Art Styles
  - Photorealistic
  - Anime
  - Vintage Victorian
  - Sketch
  - 3D Render

- 📏 Custom Image Dimensions
  - Adjustable Width
  - Adjustable Height

- ✨ Magic Enhance
  - Automatically improves prompts by adding:
    - masterpiece
    - 8k resolution
    - highly detailed
    - trending on artstation
    - unreal engine 5 render

- 🎲 Surprise Me
  - Generates random creative prompts
  - Great for inspiration

- 📥 Download Images
  - Download generated images as PNG
  - Dynamic filename based on selected art style

- ⚡ Fast and Easy UI
  - Built with Streamlit
  - Simple and responsive interface

# 📦 Installation

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Image-Studio.git
```

## 2. Open Project

```bash
cd AI-Image-Studio
```

## 3. Create Virtual Environment (Optional)

Windows

```bash
python -m venv myenv
```

Activate

PowerShell

```bash
.\myenv\Scripts\Activate.ps1
```

Git Bash

```bash
source myenv/Scripts/activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

# 📸 How to Use

1. Select an Art Style.
2. Adjust Width and Height.
3. Enter your image prompt.
4. (Optional) Enable Magic Enhance.
5. Click **Generate Image**.
6. Or click **Surprise Me!** for a random prompt.
7. Download your generated image.
